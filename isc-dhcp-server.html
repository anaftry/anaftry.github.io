DEBIAN DHCP SERVER

Perintah:

apt install isc-dhcp-server

nano /etc/dhcp/dhcpd.conf

dhcpd.conf 
authoritative;
#VLAN 10
subnet 10.0.10.128 netmask 255.255.255.128 {
  range 10.0.10.130 10.0.10.254;
  option domain-name-servers 10.0.40.238;
  option domain-name "fitri.internal";
  option routers 10.0.10.129;
  option broadcast-address 10.0.10.255;
  default-lease-time 600;
  max-lease-time 7200;
}

#VLAN 20
subnet 10.0.20.192 netmask 255.255.255.192 {
  range 10.0.20.193 10.0.20.253;
  option domain-name-servers 10.0.40.238;
  option domain-name "fitri.internal";
  option routers 10.0.20.254;
  option broadcast-address 10.0.20.255;
  default-lease-time 600;
  max-lease-time 7200;
}

#VLAN 30
subnet 10.0.30.192 netmask 255.255.255.224 {
  range 10.0.30.193 10.0.30.221;
  option domain-name-servers 10.0.40.238;
  option domain-name "fitri.internal";
  option routers 10.0.30.222;
  option broadcast-address 10.0.30.223;
  default-lease-time 600;
  max-lease-time 7200;
}

#VLAN 40 STATIC TANPA RANGE
subnet 10.0.40.224 netmask 255.255.255.240 {
  option routers 10.0.40.225;
}

Step 2

nano /etc/default/isc-dhcp-server

isc-dhcp-server 

INTERFACESv4="enp0s3"

Perintah Restart:

service isc-dhcp-server restart

service isc-dhcp-server status

Notes:

Jika dhcp server sudah berjalan abaikan dulu setting dns dulu karena domain-name-servers nya kita setting langsung pake ip debian dan fitri.internal tidak 8.8.8.8
